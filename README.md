# Personal Trainer (MVP)

MVP под курсовую: офлайн фитнес‑трекер для Android (Kotlin + Jetpack Compose + Room).

## Что реализовано в MVP
- Каталог упражнений (12 предустановленных) с описанием и группами мышц
- Создание/редактирование/удаление тренировок
  - Выбор дней недели (расписание)
  - Добавление упражнений из каталога + подходы/повторения
- Выполнение тренировки пошагово
  - Текущая тренировка, упражнение, подход
  - Общий таймер тренировки
  - Таймер отдыха между подходами (настраивается на вкладке «Сегодня»)
- История выполнений (дата/время и длительность)
- Простая статистика (за 7/30 дней, всего, топ упражнений)

## Как открыть и запустить
1. Откройте папку проекта в Android Studio (File → Open → `PersonalTrainerMVP`)
2. Дождитесь Gradle Sync (понадобится интернет для зависимостей)
3. Запустите на эмуляторе/устройстве (minSdk 26)

## Примечания
- Напоминания (уведомления по расписанию) **не включены** в MVP — добавим на следующей итерации.
- Данные хранятся локально в Room, интернет для работы приложения не нужен.
